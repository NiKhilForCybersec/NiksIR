Getting Started

L1 Section 1.0 ~8 min read

Welcome to the SOC

This guide is designed to help you become an effective SOC analyst. Whether you're brand new to security or transitioning from IT, this resource will help you build the skills you need.

How to Use This Guide

Pages are tagged by tier (L1/L2/L3). Start with L1 content, then progress as you grow. Use the search feature to find specific topics.

  1. Access: Get credentials for all SOC tools
  2. Read: Review runbooks and playbooks
  3. Shadow: Watch experienced analysts work
