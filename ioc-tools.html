OPSEC Reminder

When using public lookup tools, be aware that your queries may be logged and visible to others. For sensitive investigations, consider using private/enterprise tools or be cautious about what IOCs you submit publicly.

Multi-Purpose Tools

These tools analyze multiple IOC types and should be your first stop.

ğŸ”¬ VirusTotal

virustotal.com

The most widely used threat intelligence platform. Analyzes files, URLs, domains, and IPs against 70+ security vendors.

  • File Analysis: Upload files or search by hash (MD5, SHA1, SHA256)
  • URL Scanning: Check URLs against multiple URL scanners
  • Domain/IP Info: WHOIS, DNS, passive DNS, related files
  • Behavior Reports: Sandbox execution results for files
  • Relations: See connected IOCs (contacted IPs, downloaded files)
  • Community: Comments and votes from analysts
  • Verify if a file hash is known malware
  • Check detection ratio across vendors
  • Find related IOCs from a single indicator
  • Review behavioral analysis results
  • Determine first/last seen dates
  • Check the Detection tab first for quick verdict
  • Review Behavior tab for sandbox results
  • Use Relations to pivot to connected IOCs
  • Check Community for analyst notes
  • Note: 0 detections doesn't mean safeâ€”could be new/targeted
  • Free tier has rate limits; enterprise version available

ğŸ”— URLScan.io Deep Dive

URLScan is essential for safely analyzing suspicious URLs without visiting them.

Key Features:

  • Screenshot: See what the page looks like
  • DOM: View page source code
  • HTTP Transactions: All requests made by the page
  • Links: Extracted URLs from the page
  • Behavior: Redirects, cookies, console messages
  • Verdicts: Malicious/suspicious indicators

Tip: Use "Private" scan mode for sensitive URLs to prevent public visibility.
