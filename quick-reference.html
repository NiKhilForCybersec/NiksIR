Quick Reference Card

All Tiers Section 12.7 ~8 min read

Investigation Checklist

  • What triggered the alert?
  • What user/device is involved?
  • What time did it occur?
  • What's the process tree/parent-child?
  • Is there network activity?
  • Is this normal for this user/device?
  • Check hash/IP reputation
  • Document findings in ticket

Essential KQL

// Basic filter
TableName | where TimeGenerated > ago(24h)

// Count by field
| summarize count() by FieldName

// Search text
| where Field has "searchterm"
| where Field contains "partial"
| where Field == "exact"

// Multiple values
| where Field in ("val1", "val2")
| where Field has_any ("val1", "val2")

// Project specific fields
| project TimeGenerated, User, Action

// Top results
| top 10 by Count desc

Suspicious Indicators

Process Red Flags

  • Office ‚Üí cmd/PowerShell
  • Encoded PowerShell commands
  • Processes from Temp folders
  • Misspelled system processes
  • svchost from wrong path

Network Red Flags

  • Regular interval connections
  • Long DNS queries
  • Direct IP connections (no DNS)
  • Unusual ports outbound
  • High data upload volume

Escalation Triggers

  • Confirmed malware execution
  • Active C2 communication
  • Credential theft indicators
  • Data exfiltration
  • Lateral movement
  • Ransomware indicators
  • VIP user impacted
  • Multiple systems affected

Response Quick Actions

ScenarioImmediate Action
Active ransomwareIsolate immediately
Confirmed C2Isolate, preserve evidence
Credential theftReset password, revoke sessions
Phishing clickedCheck for payload, reset creds if entered
Malware detectedQuarantine, check spread

Reputation Check Sites

  • VirusTotal: virustotal.com (hashes, IPs, URLs)
  • AbuseIPDB: abuseipdb.com (IP reputation)
  • URLhaus: urlhaus.abuse.ch (malicious URLs)
  • Shodan: shodan.io (IP/host info)
  • MX Toolbox: mxtoolbox.com (email investigation)
